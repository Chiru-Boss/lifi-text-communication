# Li-Fi Text Communication using Laser and Arduino

## Overview

This project demonstrates wireless transmission of text data using light (Li-Fi) instead of traditional radio frequency communication.

## Objective

To design and implement a system that can transmit text messages using a laser beam and receive them using a light sensor.

## Components Used

* Arduino Uno
* Laser module (KY-008)
* Light sensor / Solar panel
* Resistors
* Connecting wires

## Working Principle

The system works by converting text data into digital signals using an Arduino.
These signals are used to turn the laser ON and OFF (modulation).

At the receiver side, a light sensor detects the laser signal and converts it back into electrical signals, which are decoded into text.
